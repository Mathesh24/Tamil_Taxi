## ✨ Features

- **Zero-commission model** overview and problem framing
- **Platform architecture** — Passenger App, Driver App, Government Dashboard, AI Dispatch
- **AI Intelligence Layer** — demand forecasting, routing, fraud detection
- **SOS Safety flow** — district-level emergency escalation
- **30-day pilot plan** with measurable success metrics
- Fully responsive design (mobile → desktop)
- Tamil Nadu government colour palette (red + yellow)
- Smooth scroll-reveal animations

## 🛠 Deployment

This site is deployed automatically to **GitHub Pages** via GitHub Actions on every push to `main`.

To set up GitHub Pages for this repo:
1. Go to **Settings → Pages**
2. Under **Source**, select **GitHub Actions**
3. Push to `main` — the workflow handles the rest
